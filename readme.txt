Generate simple Woocommerce packing slips.

== Description ==

This WooCommerce extension will generate simple packing slips from the WooCommerce order admin. 

Printing packing slips were never easy. Now it is!:) 

- Group products by order numbers.
- Simple products list.
- Display product image, name and quantity.
- Pdf format.

== Installation ==

- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress 

Or

- Upload the really-simple-packing-list-pdf (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
